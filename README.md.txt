# ELLAMIN PHOTOGRAPHY - Full Stack System

Complete photography business management with booking, galleries, and shop.

## Quick Start

1. Install Node.js 16+ from https://nodejs.org
2. Extract this zip file
3. Open terminal/command prompt in this folder
4. Run: `npm install`
5. Run: `npm start`
6. Open browser: http://localhost:5000

- Client photo galleries with downloads
- Print ordering system
- Admin dashboard
- Payment processing (Stripe)
